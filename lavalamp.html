Homebridge Lava Lamp

  • Max Chernotsky
  • Smart Home

I utilised a WiFi-enabled relay board and a Raspberry Pi to connect my lava lamp to my network.

The WiFi board shown below was programmed through the Arduino IDE. This is possible due to the onboard ESP8266 Arduino microcontroller with a built in WiFi module. I wrote Arduino code to create a wireless server from the device. This would update a simple HTML website with a 1 or 0 to indicate whether the lava lamp was on or off.

In order to control the lava lamp through Siri, an additional step was required: installing Homebridge on a Raspberry Pi. Homebridge is a device that mimics a "Home Hub", allowing any wirelessly connecting device (IoT device), to be accessed and controlled via the Home app on iOS, and by extension, Siri.

"Hey Siri, turn my lava lamp on!"

After installing the software on my Raspberry Pi, I setup a plugin that looked for the ESP8266's created server listing and adjust the listing. When a Siri call is enabled, this causes the trigger to arrive at the Homebridge software. Homebridge is then directed to the ESP8266's server and calls "/lavalamp/1" to trigger the lava lamp to turn on. This causes the relay to switch on, and the lava lamp to shine!
